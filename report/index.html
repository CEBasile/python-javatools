extension = '.html'¶
run_impl(change, entry, out)[source]¶

sets up the report directory for an HTML report. Obtains the top-level Cheetah template that is appropriate for the change instance, and runs it.

The cheetah templates are supplied the following values:
  • change - the Change instance to report on
  • entry - the string name of the entry for this report
  • options - the cli options object
  • breadcrumbs - list of backlinks
  • javascripts - list of .js links
  • stylesheets - list of .css links

The cheetah templates are also given a render_change method which can be called on another Change instance to cause its template to be resolved and run in-line.
